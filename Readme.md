## Usage

1. Download the repository

```
$ git clone https://github.com/NLeflohic/expensify-app.git
$ cd expensify-app
```

2. Installation

```
$ yarn install
# or
$ npm install

```

3. Build

```
$ yarn build:dev
# or
$ npm run build:dev
```

4. Run

```
$ yarn dev-server
# or
$ npm run dev-server
```

Open your favorite web-browser and fill the url : localhost:8080/
